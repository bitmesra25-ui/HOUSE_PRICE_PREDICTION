# House Price Prediction

This project predicts house prices using Simple Linear and Multiple Linear Regression.

## Technologies
- Python
- Pandas
- Seaborn
- matploylib
- NumPy
- Scikit-learn
- Jupyter Notebook

## Features
- Data preprocessing
- Train-test split
- Model training
- Model evaluation (MSE, R2)
- User input prediction

## Result
The model was evaluated using MSE and R2 score and achieved good accuracy.
